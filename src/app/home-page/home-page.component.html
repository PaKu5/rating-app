Funktionalität

Übsericht

Übersicht aller Händler und Anzeige der aktuellen Punkte.
Momentan ist hier eine "Fake"-Datenbank dahinter. Das bedeutet, dass bei jedem Neuladen der Seite wieder die ursprünglichen Daten angezeigt werden. Diese können aber jederzeit angepasst werden.

Punkte

Übsericht der Händler, deren Gesamtpunkte, ausgegebene Punkte und aktueller Punktzahl.

Login

Benutzername:admin

Nach dem Login muss man erst einmal auf eine andere Seite wechseln, damit man die neuen Funktionen sehen kann.

Funktionen

Admin-Seite
In der Toolbar erscheint ein zusätzlicher Punkt Admin, hier können Händler erstellt, bearbeitet und gelöscht werden.
Übersicht
Hier erscheint zusätzlich ein Button, mit dem eine Bewertung für die Händler abegegeben werden kann.
Punkte
Hier kann zusätzlich die Anzahl der ausgegebenen Punkte eingestellt werden.
