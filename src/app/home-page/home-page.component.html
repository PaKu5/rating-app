Funktionalität

Übersicht

Übersicht aller Kundenbetreuer und Anzeige der aktuellen Punkte.
Zurzeit ist hier keine echte Datenbank hinterlegt. Aus diesem Grund werden jegliche Vorgänge nicht dauerhaft gespeichert.

Punkte

Übersicht der Kundenbetreuer, deren Gesamtpunkte, ausgegebene Punkte und aktueller Punktzahl.

Funktionen

Admin-Seite
In der Toolbar erscheint ein zusätzlicher Punkt Admin, hier können Kundenbetreuer erstellt, bearbeitet und gelöscht werden.
Übersicht
Hier erscheint zusätzlich ein Button, mit dem eine Bewertung für die Kundenbetreuer abegegeben werden kann.
Punkte
Hier kann zusätzlich die Anzahl der ausgegebenen Punkte eingestellt werden.
